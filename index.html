1. Поключение без параметров

<script>
	$(".slider-container").sliderUi();
</script>

2. Расширенные настройки

По умолчанию плагин использует анимацию css3 (рекомендуется).

<script>
	$(".slider-container").sliderUi({
		speed: 500,
		cssEasing: "ease-in-out" //описание easing для css анимации
	});
</script>

Чтобы использовать js анимацию, в параметрах необходимо отключить опцию cssAnimate

<script>
	$(".slider-container").sliderUi({
		cssAnimate: false,
		jsEasing: "linear" //не обязательно
	})
</script>

Параметры

Список всех доступных опций и их значения по умолчанию

controlShow: true, // показывать навигацию внизу
arrowsShow:  true, // показывать вперед|назад навигацию
autoPlay:    true, // автоматическое перелистывание изображений
delay:       3000, // задержка перед перелистыванием
caption:     true, // показывать описание
speed:       500, // скорость анимации
cssAnimate:  true // анимации с помощью CSS(рекомендуется)
cssEasing:   "ease-out" // функция плавности анимации с помощью CSS
jsEasing:    "linear" // функция плавности анимации с помощью JS
